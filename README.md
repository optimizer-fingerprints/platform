# Optimizer Fingerprints

A [small artifact](https://www.optimizers.space/) to look at what optimizers actually do to updates during training. It logs a bunch of update geometry metrics over [nanoGPT training runs](https://github.com/KellerJordan/modded-nanogpt/tree/master/records/track_3_optimization), then lets you poke through them by optimizer, layer, and time.
